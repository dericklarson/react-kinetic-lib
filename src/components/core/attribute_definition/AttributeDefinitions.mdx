## Attribute Definition Form

### Props

<Props of={AttributeDefinitionForm} />

### Examples

#### Simple implementation of a `AttributeDefinitionForm`.

<Playground>
  <AttributeDefinitionForm attributeType="spaceAttributeDefinitions" />
</Playground>

Because no name was passed, submitting this form will create a new Space Attribute Definition

#### Control the Layout of a `AttributeDefinitionForm`.

<Playground>
  <AttributeDefinitionForm attributeType="spaceAttributeDefinitions">
    {({ form }) => (
      <div>
        <h5>My Form Header</h5>

      </div>
    )}
  </AttributeDefinitionForm>
</Playground>

Because no name was passed, submitting this form will create a new Space Attribute Definition
