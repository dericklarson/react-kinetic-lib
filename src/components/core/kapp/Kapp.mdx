## Kapp Form

### Props

<Props of={KappForm} />

### Examples

#### Simple implementation of a `KappForm`.

<Playground>
  <KappForm kappSlug="services" />
</Playground>

Because no name was passed, submitting this form will create a new Form

#### Control the Layout of a `KappForm`.

<Playground>
  <KappForm kappSlug="services">
    {({ form }) => (
      <div>
        <h5>My Form Header</h5>
        {form}
      </div>
    )}
  </KappForm>
</Playground>

Because no name was passed, submitting this form will create a new Space Attribute Definition
