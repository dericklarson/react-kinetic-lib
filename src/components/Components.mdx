# Kinetic React Components

#### Provided in this library are a number of components that allow can be used within a project to enable speedy development within the Kinetic Platform.

- Core Components: Interact with the Core Service
- Discussions Components: Interact with the Discussions Service
- Form Components: A generic component that can be used for rendering forms
- Table Components: A generic component that can be used for rendering tables
