# Kinetic React Components

#### Provided in this library are a number of components that allow can be used within a project to enable speedy development within the Kinetic Platform.

- [Core Components](#/Core) that interact with the Core Service
- [Discussions](#/Discussions) that interact with the Discussions Service
- [Form Components](#/Forms): A generic component that can be used for rendering forms
- [Table Components](#/Tables): A generic component that can be used for rendering tables
